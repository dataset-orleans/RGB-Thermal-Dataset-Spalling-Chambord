# RGB-Thermal-Dataset-Spalling-Chambord
Dataset for multimodal semantic segmentation of spalling and pre-spalling defects on stone surfaces of Château de Chambord, France. 

The dataset contains 73 aligned RGB-thermal image pairs, with RGB images of 320 × 240 pixels and corresponding thermal data stored in NumPy (.npy) format. 

Pixel-level annotations are provided for the identification and segmentation of stone deterioration areas, including spalling and pre-spalling defects. The dataset includes 10 manually annotated defect regions.
